# Screenshots Directory

This directory contains screenshots of the application's user interface.

## Required Screenshots

Please add the following screenshots to this directory:

1. **home-page.png** - Landing page with explanation, video access, and real-life examples
2. **clustering-side-by-side.png** - Scatter plot and dendrogram displayed side by side
3. **clustering-top-bottom.png** - Scatter plot and dendrogram displayed in top-bottom layout
4. **video-modal.png** - Full-screen video explanation modal
5. **real-life-examples.png** - Examples of clustering applications in various domains
6. **control-panel.png** - Control panel with dataset selection, algorithm options, and playback controls
7. **dendrogram-detail.png** - Detailed view of the dendrogram with all data points clearly visible
8. **scatter-plot.png** - Scatter plot showing clustered data points with unique colors

## How to Take Screenshots

1. Run the application: `npm run dev`
2. Navigate to `http://localhost:3000`
3. Take screenshots of each view
4. Save them in this directory with the names listed above
5. Ensure screenshots are clear and showcase the key features

## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: At least 1920x1080 for best quality
- **File Size**: Keep under 2MB per image for optimal loading
- **Naming**: Use lowercase with hyphens (e.g., `home-page.png`)
